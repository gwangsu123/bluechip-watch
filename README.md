## 아이폰 앱으로 설치하는 방법

### 1단계 — GitHub Pages로 무료 호스팅

1. [github.com](https://github.com) 접속 → 계정 없으면 무료 가입
2. 우측 상단 **+** → **New repository**
3. Repository name: `bluechip-watch` (또는 원하는 이름)
4. **Public** 선택 → **Create repository**
5. **uploading an existing file** 클릭
6. 이 폴더의 파일 4개 모두 드래그 업로드:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `README.md`
7. **Commit changes** 클릭
8. **Settings** 탭 → 왼쪽 **Pages** → Source: **Deploy from a branch** → Branch: **main** → **Save**
9. 몇 분 후 아래 주소로 접속 가능:
   ```
   https://[내GitHub아이디].github.io/bluechip-watch
   ```

### 2단계 — 아이폰 홈 화면에 추가

1. 아이폰 **사파리**로 위 주소 접속
2. 하단 공유 버튼 탭 (□↑)
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **추가** 탭
5. 홈 화면에 앱 아이콘 생성 완료 🎉

> ⚠️ 반드시 **사파리**에서 해야 합니다. 크롬/네이버앱 등에서는 홈 화면 추가가 안 됩니다.

## 종목 데이터 업데이트 방법

`index.html` 파일 안에서 `const ST=[` 부분을 찾아  
각 종목의 등락률(4번째 숫자)을 수정하면 됩니다.

```js
// [시총순위, 종목코드, 종목명, 등락률%, 섹터ID, 업종]
[1,'005930','삼성전자', 3.87, 'semi', '반도체'],
```

수정 후 GitHub에 다시 업로드하면 자동 반영됩니다.
